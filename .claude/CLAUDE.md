# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Quick Start

- This is a multi-project workspace (`GL-SRE/`, `GLChat/`). Always `cd` into the target project before running commands.
- Check `.claude/rules/` for detailed rules on workspace structure, project commands, tech patterns, and task workflow.
- Check `.claude/.claude-memory.md` for active task context and session resumption state.
- Check `tasks/` for task history and progress tracking.
- Check `.claude/skills/` for available skills (e.g., `/push-issue`, `/new-task`).
- Check `.claude/docs/` for skill usage guides.
- Check project-specific instruction files before deep-diving into any subproject.
